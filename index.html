About

Mapping community gardens from the NYC Open Data GreenThumbs datasets. The project will eventually become an overview of urban greenery datasets from the NYC Open Data portal.

Methodology

NYC OpenData API's + MapboxglJS + HTML/CSS/JS

Disclaimer

This project is done for an Information Visualization class as part of my Master's in Information Experience Design at Pratt Institute. It leverages NYC Open Data but has no affiliation with the city.
